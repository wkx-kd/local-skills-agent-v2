# 营销素材通用设计原则

## 茶饮产品摄影提示词模板

### 核心要素（每张图的提示词都应包含）
- **产品描述**: 饮品的外观、颜色、层次、配料
- **光影**: "暖色自然光"、"侧逆光"、"金色光斑"
- **质感关键词**: "奶盖缓缓流动"、"冰块晶莹剔透"、"杯壁凝结水珠"、"桂花瓣轻浮于茶面"
- **风格关键词**: "新中式美学"、"国潮设计"、"高级感"、"ins风"
- **负面提示词**: 统一使用以下 negative_prompt：
  "低分辨率，低画质，肢体畸形，手指畸形，画面过饱和，蜡像感，人脸无细节，过度光滑，画面具有AI感，构图混乱，文字模糊扭曲，廉价感，塑料质感"

### 提示词结构公式
```
[场景描述] + [产品细节] + [品牌元素] + [光影氛围] + [风格定义]
```

示例：
```
秋日午后的新中式茶馆，一杯精致的桂花乌龙奶茶置于深木色托盘上，
杯身透明玻璃清晰展现三层渐变——底部琥珀色乌龙茶、中层丝滑奶白、
顶部蓬松奶盖上点缀金黄桂花瓣。杯旁散落几朵干桂花和一枚朱砂色印章。
暖色侧逆光透过竹帘洒入，在深墨绿色背景上勾勒出柔和光晕。
整体风格：新中式美学，高级质感，商业摄影级画质。
```

## 批量生成策略
生成10张图时，确保多样性：
1. 2张 产品特写（不同角度）
2. 2张 场景氛围（不同场景）
3. 2张 文字海报（不同文案）
4. 2张 手持/互动场景
5. 1张 原料展示（桂花、茶叶）
6. 1张 品牌概念图（抽象/意境）
